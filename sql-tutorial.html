Interactive SQL Tutorial

This page runs SQLite entirely inside your browser. You can write SQL, run it from JavaScript, inspect the result sets, and then check whether your query or data change matches the exercise.

The only external dependency is sql.js, loaded from a CDN. Everything else on this page is plain HTML, CSS, and readable JavaScript.

Important Join Note

You cannot move a JOIN clause before FROM in SQL syntax. What is often true is that inner joins can be written in different table orders and still return the same rows if you preserve the same join conditions.

That does not hold in general for outer joins such as LEFT JOIN. One of the exercises below has you reorder inner joins and confirm that the result set stays the same.
